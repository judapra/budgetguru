# **App Name**: Budget Guru

## Core Features:

- Financial Dashboard: A comprehensive dashboard providing an overview of the user's financial status for both personal and business finances, displayed in separate tabs.
- Income and Expense Tracking: Automated tracking of income and expenses, categorized for easy analysis and reporting.
- Budget Creation and Management: Allow users to create and manage budgets for different categories (e.g., rent, groceries, utilities).
- Real Estate Management Tool: Tool which allows users to input and organize real estate financial information and output useful reports about their properties' expenses, income, equity and ROI.
- Report Generation: Generate reports on income, expenses, and budget performance for different time periods.

## Style Guidelines:

- Primary color: Soft teal (#71C2D8) to promote a sense of calm and trustworthiness.
- Background color: Light off-white (#F2F4F3) for a clean and unobtrusive backdrop.
- Accent color: Muted blue (#5A8291) to highlight key actions and information without causing distraction.
- Headline font: 'PT Sans', sans-serif. Body font: 'Literata', serif.
- Use clean and professional icons for different categories of transactions and financial metrics.
- A tab-based layout with clear sections for personal finance, business finance, and real estate management.
- Subtle animations for transaction updates and report generation to enhance user experience without being distracting.
